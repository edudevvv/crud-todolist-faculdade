# 🗓️ Gerenciador de Tarefas
> O sistema foi desenvolvidor com intuito de auxiliar nas tarefas diárias.

# 🛠️ Principais Funcionalidades:
> 1. Usuários: Login, Registro e Dashboard com Tarefas (em desenvolvimento)
> 2. Tarefas: Criar, Listar, Atualizar, Deletar (em desenvolvimento)
> 3. Acompanhamento em tempo real. (em desenvolvimento)

# ⚙️ Tecnologias utilizadas:
> - Back-End: Spring Boot, MongoDB
> - Front-End: HTML, CSS, BOOTSTRAP

# 🗒️ Desenvolvimento das funcionalidades:
```
    1. Usuários: 
    > Login - 90% (⌛)
    > Registro - 100% (✅)
```
